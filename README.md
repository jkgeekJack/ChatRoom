# ChattingRoom
这是一个java语言实现的B/S结构聊天室

##软件功能
1. 登录注册
2. 点对点聊天
3. 聊天记录
4. 好友系统
5. 上线下线提醒
6. 文件传输

##软件架构
1. 通讯采用TCP连接
2. 数据库采用MySQL
3. 数据库框架采用Hibernate
4. UI采用Swing
